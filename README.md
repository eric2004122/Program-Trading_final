## 核心回測邏輯

兩個版本皆採用相同的核心策略：

*   **交易標的**: `0050.TW` (元大台灣50 ETF)
*   **買進訊號**: MACD 黃金交叉。
*   **賣出訊號**: MACD 死亡交叉、固定比例停損 (`-5%`)、固定比例停利 (`+10%`)。
*   **交易成本**:
    *   **手續費**: `0.1425%` (買進與賣出)
    *   **交易稅**: `0.1%` (僅賣出)
